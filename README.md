# Animal Doku Puzzle Game

A relaxing mobile logic puzzle inspired by meowDoku, with interchangeable animal themes.

## Requirements

- **Xcode 15+** (Swift 5.9+)
- **iOS 17.0+** deployment target (SwiftUI, `@Observable`)
- iPhone, portrait orientation only (MVP)

## Getting Started

1. Open `AnimalDoku.xcodeproj` in Xcode.
2. Select an iPhone simulator (e.g. iPhone 15).
3. Build and run (⌘R).

From the command line:

```bash
xcodebuild -scheme AnimalDoku -destination 'platform=iOS Simulator,name=iPhone 16' build
```
