# Sound effects (P4.5)

Calm, short clips played by `SoundService`:

| File | Event |
|------|--------|
| `place.wav` | Animal placed (or hinted) |
| `remove.wav` | Animal removed |
| `win.wav` | Puzzle completed |

Replace these with designer-authored assets anytime — keep the same file names.
Playback respects Settings → Sound Effects and the device silent switch (`.ambient` audio session).
